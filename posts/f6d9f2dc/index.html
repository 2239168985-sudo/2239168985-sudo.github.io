下一步（按优先级排序）

1. 拿真实新增图像（最高优先级）

第三篇已经论证：瓶颈是场景多样性。两条路：

  • 实地补拍：针对真实河道 / 湖泊场景补充拍摄；
  • 引入公开数据集：IWHR_AI_Lable_Floater_V1（Scientific Data 2025，3000 张内陆水域漂浮物图像），做合并训练或域适应。

2. 标注草稿人工过一遍

审计产出的清标注草稿（见第四篇）逐张人工确认后转正。

3. 主攻 grass 类

真实口径下 grass mAP50 ≈ 0.49 是全局瓶颈。策略组合：新场景数据 + 多背景 copy_paste + 语义判别层面的针对性设计。

4. 成果发布

  • GitHub 仓库公开（SSH 走 22 端口方案已验证可写，待公钥绑定账号）；
  • CSDN 博客同步发布（待登录态）。

可复现性（附录）

为方便他人（和未来的自己）复现，环境与口径约定如下：

项 内容
训练入口 项目根目录下各 train_xxx.py 脚本，conda 环境内直接 python train_xxx.py
混合精度 自定义模块实验统一 amp=False（避免数值不稳定）
环境 ultralytics 8.4.90（源码）/ PyTorch 2.6.0+cu124 / Python 3.13 / RTX 4060 8GB
评测口径 统一用官方 test 1200 图评测；严格区分「旧泄漏口径」与「真实口径」
数据划分 官方 train 600 / val 600 / test 1200，seed=42

完整实验台账见仓库内《实验精度汇总.md》。
